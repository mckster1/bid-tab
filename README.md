v9.1 Sort Patch (FIXED)

Important:
- There is ONLY ONE .bas file in this patch.
- That single module contains TWO public macros you can assign to buttons:
    SortBidders_ByBaseBid_v9_1
    SortBidders_ByAdjustedBid_v9_1

Install:
1) ALT+F11
2) In Project Explorer -> Modules, remove any old "TSC_v9_1_SortBidders" module
3) File -> Import File -> import TSC_v9_1_SortBidders.bas
4) Debug -> Compile VBAProject

Assign:
- Sort Base button -> SortBidders_ByBaseBid_v9_1
- Sort Adjusted button -> SortBidders_ByAdjustedBid_v9_1
